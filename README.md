# Execute Program

Generate java code from jflex file on command line:
```PS
jflex -nobak -q src/lexer.flex
```

Compile java file:
```PS
javac -d bin src/lexer.java
```

Execute java class:
```PS
java -cp bin lexer input.txt
```
